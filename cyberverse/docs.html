API Documentation

Complete guide to integrating CyberVerse NFT functionality into your applications

v2.0

Introduction

Welcome to the CyberVerse API documentation. Our REST API allows you to integrate NFT minting, trading, and marketplace functionality into your applications. Built on Ethereum and compatible with all ERC-721 and ERC-1155 standards.

üìò Base URL:
https://api.cyberverse.io/v2

Key Features

  • Full NFT lifecycle management (mint, transfer, burn)
  • Marketplace integration (listing, buying, offering)
  • Real-time price data and analytics
  • Wallet connection and transaction signing
  • Collection and metadata management
  • WebSocket support for live updates

Authentication

All API requests require authentication using an API key. You can generate your API key from your developer dashboard.

‚ö†Ô∏è Security Warning:
Never expose your API keys in client-side code or public repositories. Keep them secure on your backend server.

Obtaining an API Key

  1. Log in to your CyberVerse account
  2. Navigate to Settings ‚Üí Developer
  3. Click "Generate API Key"
  4. Copy and securely store your key

Making Authenticated Requests

Include your API key in the request header:

JavaScript 
const response = await fetch('https://api.cyberverse.io/v2/nfts', {
  headers: {
    'Authorization': 'Bearer YOUR_API_KEY',
    'Content-Type': 'application/json'
  }
});
Python 
import requests

headers = {
    'Authorization': 'Bearer YOUR_API_KEY',
    'Content-Type': 'application/json'
}

response = requests.get('https://api.cyberverse.io/v2/nfts', headers=headers)

API Endpoints

NFT Endpoints

GET /nfts

Retrieve a list of NFTs with optional filtering and pagination.

Query Parameters

Parameter Type Description
page integer Page number (default: 1)
limit integer Items per page (default: 20, max: 100)
category string Filter by category (art, music, gaming, etc.)
min_price number Minimum price in ETH
max_price number Maximum price in ETH

Example Response

JSON 
{
  "success": true,
  "data": [
    {
      "id": "0x123...",
      "name": "Cyber Punk #4521",
      "description": "Unique cyberpunk NFT",
      "price": 2.5,
      "currency": "ETH",
      "image_url": "https://...",
      "creator": "0xabc..."
    }
  ],
  "pagination": {
    "page": 1,
    "limit": 20,
    "total": 500
  }
}
POST /nfts/mint

Mint a new NFT on the blockchain.

Request Body

Field Type Description
name * string NFT name
description string NFT description
image * string Image URL or base64 data
properties object Metadata properties
royalty number Royalty percentage (0-50)
POST /nfts/{id}/transfer

Transfer an NFT to another wallet address.

Marketplace Endpoints

POST /marketplace/list

List an NFT for sale on the marketplace.

POST /marketplace/buy

Purchase a listed NFT from the marketplace.

DELETE /marketplace/delist/{id}

Remove an NFT listing from the marketplace.

Quick Start Example

TypeScript 
import { CyberVerse } from '@cyberverse/sdk';

// Initialize the client
const client = new CyberVerse({
  apiKey: process.env.CYBERVERSE_API_KEY
});

// Fetch NFTs
const nfts = await client.nfts.list({
  category: 'art',
  limit: 10
});

// Mint a new NFT
const newNFT = await client.nfts.mint({
  name: 'My First NFT',
  image: 'ipfs://...',
  properties: {
    rarity: 'rare'
  }
});

Rate Limits

To ensure fair usage and platform stability, API requests are rate-limited:

Plan Requests/Minute Requests/Day
Free 60 10,000
Pro 300 100,000
Enterprise Unlimited Unlimited
Rate Limit Headers:
X-RateLimit-Limit: Total requests allowed
X-RateLimit-Remaining: Requests remaining
X-RateLimit-Reset: Time when limit resets

Error Handling

The API uses standard HTTP status codes to indicate success or failure:

Code Status Description
200 OK Request successful
400 Bad Request Invalid request parameters
401 Unauthorized Invalid or missing API key
404 Not Found Resource not found
429 Too Many Requests Rate limit exceeded
500 Internal Server Error Server error

Error Response Format

JSON 
{
  "success": false,
  "error": {
    "code": "INVALID_API_KEY",
    "message": "The provided API key is invalid",
    "details": {}
  }
}

Code Examples

Complete Minting Example

JavaScript 
async function mintNFT() {
  const response = await fetch('https://api.cyberverse.io/v2/nfts/mint', {
    method: 'POST',
    headers: {
      'Authorization': 'Bearer YOUR_API_KEY',
      'Content-Type': 'application/json'
    },
    body: JSON.stringify({
      name: 'Cyber Punk #1',
      description: 'A unique cyberpunk NFT',
      image: 'ipfs://QmXyz...',
      properties: {
        rarity: 'legendary',
        power: 95
      },
      royalty: 10
    })
  });

  const data = await response.json();
  console.log('Minted NFT:', data);
}
